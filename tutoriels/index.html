Se connecter

Pour vous connecter à l'interface d'administration de votre site, il faut vous rendre à l'adresse de celui-ci et ajouter /perch à la fin.

Par exemple, si le nom de votre site est www.psychologue.org, il vous faudra entrer dans votre navigateur www.psychologue.org/perch

Il vous suffit alors d'entrer vos identifiants.

voir la démo

Accéder à une page pour éditer son contenu

Il est très simple d'ajouter du contenu à votre site via votre interface d'administration.

Avant tout, accédez à la page que vous souhaitez éditer dans votre interface. Pour cela, il suffit de cliquer sur le nom de la page.
